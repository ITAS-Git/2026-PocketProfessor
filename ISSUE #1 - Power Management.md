### ISSUE #1: Power Management: Reliable On/Off Control
Label: User Story

**User Story:**
As a user, I want the Pocket Professor to turn on and off reliably so that I can control when it is active and available for use.

- [ ] Install power management hardware components (18650 battery system, wiring)
- [ ] Implement a safe software shutdown mechanism in Python
- [ ] Test power on/off cycling for reliability and consistency
- [ ] Document power setup steps in the hardware folder

Dependencies: None, this is a foundational task.
